# Ukrainian Language Exam Preparation Web Platform

## Overview
This project is an **educational web-based platform** created as a **graduation academic project**.  
Its goal is to help **10th-grade students** prepare for Ukrainian language exams through **structured learning materials and interactive knowledge testing**.

The platform combines **theoretical content**, **exam tickets**, and **interactive quizzes**, providing students with a clear, user-friendly, and accessible learning experience.

---

## Project Goals
- Provide an **online learning tool** for Ukrainian language exam preparation
- Support recent updates in Ukrainian language rules and orthography
- Combine **learning and self-assessment** in one platform
- Apply **UX/UI principles** to improve usability and engagement

---

## Key Features

### Learning Section
- Structured theoretical materials presented as **exam tickets**
- Clear explanations with examples and highlighted exceptions
- Easy navigation between topics using animated buttons and menu

### Testing Section
- Interactive quizzes divided into thematic sections
- Multiple-choice questions with instant visual feedback
- Final results page with score calculation
- Recommendations for revising topics based on test results

### User Experience
- Intuitive navigation with a **hamburger menu**
- Responsive layout for desktop and mobile devices
- Visual hierarchy and highlighted key information
- Animated buttons and interface elements for better interaction

---

## Design & UX
- Interface designed in **Figma**
- Focus on:
  - Usability
  - Accessibility
  - Clear information structure
  - User-centered design
- Layout planned using wireframes and page structure diagrams
