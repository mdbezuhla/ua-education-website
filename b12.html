    Білет 12

    12.1. Подовження приголосних перед я, ю, є, ї.

    Відбувається подовження

  1. Життя, обличчя, насіння.

    2. Іменники другої відміни середнього роду на я, які закінчуються на приголосні л, д, т, з, ц, ч, ш, ж, н (можна запам’ятати «Де ти з’їси цю чашу жиру», але без літери р).

    Запам’ятати : Ілля, суддя, рілля, баддя, стаття, спросоння, навмання, зрання, попідтинню, попідвіконню.

    В іменниках 3 відміни ж. р. в орудному відмінку перед ю відбувається подовження :

    Н.в. Ніч Піч Подорож
    Р.в. Ніччю Піччю Подорожжю

  2. 3. Якщо в родовому відмінку множини є закінчення ів, то подовження зберігається. Якщо закінчення ів відсутнє, то подовження не зберігається.

    3. Н.в. Знання Почуття
    Р.в. Знань Почуттів

  3. Перед я, ю, у формах дієслова лити : ллє, наллю, виллєш, ллємо.

Не відбувається подовження

  1. Щастя, жовчю, честю, Керчю.

    2. Якщо збігаються дві різні приголосні.

  2. Сузір’я, матір’ю.

    3. Якщо слово пишеться з апострофом.

  3. Кошеня, цуценя, теля.

    4. В іменниках третьої відміни(маленькі істоти).

    Запам’ятати : попадя, кутя, третя.

12.2. Кличний відмінок іменників І відміни.

    У кличному відмінку однини іменники І відміни мають такі закінчення :

  1. закінчення -о- мають іменники твердої групи : сестра-сестро, правда-правдо;

  2. закінчення -е- мають іменники м’якої та мішаної групи : земля-земле, душа-душе;

  3. закінчення -є- мають іменники м’якої групи, після голосного та апострофа : Марія-Маріє, сім’я-сім’є.

    4. АЛЕ :

    • Ілля-Ілле;

  4. закінчення -ю- мають деякі іменники м’якої групи, пестливі іменники : бабуся-бабусю, Галя-Галю;

  5. у звертаннях, що складаються з імені та по батькові, обидва слова мають закінчення кличного відмінка : Олено Володимирівно, Тамаро Валентинівно.
